# actions-node

Composite Github Action to provides opinionated NodeJS steps

## Steps

- Checkout
- Setup Node.js, with given package manager and version retrieve from `.nvmrc`
- Install Dependencies
- Lint
- Build
- Test
- Code coverage reporting if `coverage` is enabled
